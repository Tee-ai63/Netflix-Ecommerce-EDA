# Data Cleaning & Exploratory Data Analysis — E-commerce & Netflix

AnalystLab Africa — Data Analytics Track — Week 1–2 Assignment

This repo contains the same end-to-end data cleaning and EDA process applied to two
different datasets: a UK-based online retail transaction dataset, and the Netflix
Titles catalog. Each folder is self-contained with the cleaned data, the full
Jupyter notebook (dataset understanding → cleaning → EDA → visualization →
insights), and a one-page summary report.

## Repo Structure

```
├── ecommerce/
│   ├── Ecommerce.ipynb                 # Full analysis notebook
│   ├── OnlineRetail_Cleaned.zip        # Cleaned dataset (zipped — unzip to use)
│   └── Ecommerce_Summary_Report.docx   # One-page summary report
│
├── netflix/
│   ├── Netflix.ipynb                   # Full analysis notebook
│   ├── netflix_titles_cleaned.csv      # Cleaned dataset
│   └── Netflix_Summary_Report.docx     # One-page summary report
```

> **Note:** `OnlineRetail_Cleaned.zip` is compressed to stay under GitHub's file
> size limits. Unzip it locally to get `OnlineRetail_Cleaned.csv`.

## Datasets

- **Online Retail (E-commerce):** [Kaggle — UCI Online Retail Dataset](https://www.kaggle.com/datasets/vijayuv/onlineretail) — transaction-level purchase records from a UK-based online retailer, Dec 2010 onward.
- **Netflix Titles:** [Kaggle — shivamb/netflix-shows](https://www.kaggle.com/datasets/shivamb/netflix-shows) — Netflix's catalog of Movies and TV Shows.

## Process Applied to Both Datasets

1. **Dataset Understanding** — shape, data types, feature classification, primary key identification
2. **Data Cleaning** — missing values, duplicates, standardization (dates, text, column names, data types), data validation
3. **Exploratory Data Analysis** — summary statistics and 5+ targeted explorations per dataset
4. **Data Visualization** — 5+ charts per dataset, each with title, labels, and findings
5. **Insights** — 5 key, analysis-backed insights per dataset
